About

道路公社主任の視点を活かし、技術局で開発を担う

Tools

道路公社ツールサイト

rc-tools は、道路公社の作業を支援するためのツールサイトです。URL変換、座標確認、折れ線計画、ペイント量計算、完成予定日の推定などを扱います。

Bots

Discord Bot

北レーダーbot、沖ノ鳥島開発bot、OkiMail を制作しています。安定運用だけでなく、内容変更のしやすさも見ながら、ローカル実行と Cloudflare Workers を使い分けています。

Infra

wplaceoki.com

wplaceoki.com を所有し、GitHub Pages、Cloudflare Tunnel、OkiMail のメールアドレス発行、Minecraft サーバー向けの NS 移譲など、ドメインとサブドメインの運用も行っています。

Systems

Cloudflare を中心に開発

北レーダーbotは、内容変更のしやすさや小回りを重視してローカル環境で実行しています。一方で、rc-tools、沖ノ鳥島開発bot、OkiMail などは Cloudflare Workers を使って構築しています。

wplaceoki.com のドメインは、OkiMail のメールアドレス発行にも使っています。サブドメインは、GitHub Pages の公開先として使うだけでなく、Cloudflare Tunnel や、Aternos の Minecraft サーバーへ移譲する用途にも使っています。

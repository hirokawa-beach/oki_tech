# OkiMail よくある質問

source: bot-spec
updated: 2026-04-26
category: bot

## 自分でアドレスを自由に作れますか？

できません。

次のチャンネルで、決められた形式で申請してください。

https://discord.com/channels/1405845560413651026/1463143161437229261

いわゆる捨てメアドを作りたい場合も、その旨を同じチャンネルで申し出てください。

## OkiMail から返信や送信はできますか？

できません。

OkiMail は、受信したメールを Discord に転送する機能のみ対応しています。

## 画像以外のファイルも届きますか？

はい。

画像は、本文埋め込み・通常添付のどちらも転送可能です。

ソースコードや ZIP などのファイルも受信可能です。

ただし、中身には十分注意し、信頼できる送信者のもののみ開くようにしてください。

サイズ制限により、省略される場合もあります。

## 添付が省略されるのはどんな場合ですか？

Cloudflare の合計 25MiB 制限、または Discord の単体 10MiB 制限を超える場合などです。

送信時・受信時ともに、Cloudflare の合計 25MiB 制限が適用されます。

ファイル単体が 10MiB を超える場合、一部またはすべてのファイルが省略されることがあります。

## 一部のメールが文字化けするのはなぜですか？

メールの形式やエンコード方式が特殊な場合、デコード処理の都合で文字化けが起きることがあります。

## OkiMail 登録完了のお知らせが届きません

DM や年齢設定などの影響で届かない場合があります。

その場合は、OkiMail Bot の「アプリを追加」を押したうえで、次のチャンネルにて再発行を申し出てください。

https://discord.com/channels/1405845560413651026/1463143161437229261

OkiMail Bot のユーザー ID は `1446422325594554482` です。

## サーバーを抜けても利用できますか？

できません。

OkiMail Bot を導入しているサーバーに加入していないと、Bot の権限が届かないため、メールの受信ができなくなります。

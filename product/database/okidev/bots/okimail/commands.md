# OkiMail コマンド

source: bot-spec
updated: 2026-04-26
category: bot

## 概要

OkiMail は、送信元アドレス単位のブロック機能を備えています。

ブロックはユーザー ID 単位で管理されるため、複数の OkiMail アドレスを持っていても、同一ユーザーとして一括で適用されます。

ブロックした送信元からのメールは、DM に届かず通知もされません。

## /block

`/block` は、指定した送信元メールアドレスからの受信を拒否するコマンドです。

```text
/block <メールアドレス>
```

## /unblock

`/unblock` は、指定した送信元メールアドレスのブロックを解除するコマンドです。

```text
/unblock <メールアドレス>
```

## /id

`/id` は、自分の OkiMail アドレス一覧を確認するコマンドです。

```text
/id <ユーザーID>
```

このコマンドは、本人または管理チャンネルのみ有効です。

ユーザー ID を確認するには、Discord のユーザー設定から詳細設定を開き、開発者モードを ON にしてください。

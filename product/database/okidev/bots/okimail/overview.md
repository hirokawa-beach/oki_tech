# OkiMail

source: bot-spec
updated: 2026-04-26
category: bot

## 概要

OkiMail は、発行したメールアドレスに届いたメールを Discord の DM に転送する Bot です。

メール本文を抽出して Discord で表示可能な形式に変換し、利用者の DM へ送信します。

本文だけでなく、送信されたメールファイル（`.eml`）も添付されます。

画像などの添付ファイルも転送できます。

ただし、ファイル容量が大きい場合は、メールファイルや添付の一部が省略される場合があります。

## 主な機能

- OkiMail アドレス宛てメールの自動受信
- メール本文の抽出とデコード
- Discord DM への自動転送
- `.eml` メールファイルの添付
- 画像や ZIP、ソースコードなどの添付ファイル転送
- 送信元メールアドレス単位のブロック

## ブロック機能

OkiMail には、送信元アドレス単位のブロック機能があります。

ブロックした送信元からのメールは、DM に届かず通知もされません。

ブロック機能はユーザー ID 単位で管理されます。

そのため、複数の OkiMail アドレスを所持している場合でも、同一ユーザーとして一括でブロックが適用されます。

## 重要な位置づけ

OkiMail は、遊び用途や検証用途を想定したメール転送 Bot です。

セキュリティ上の欠陥が存在する可能性があるため、メインのメールアドレスとしての利用は推奨されません。

# OkiMail 利用の流れ

source: bot-spec
updated: 2026-04-26
category: bot

## 発行依頼

OkiMail アドレスの発行を希望する場合は、次のチャンネルで申し出てください。

https://discord.com/channels/1405845560413651026/1463143161437229261

依頼時には、次の内容を伝えてください。

- 発行したい名前
- 希望する所属（局などのロール）

例:

```text
example.tech@wplaceoki.com
```

## 利用手順

1. 発行依頼チャンネルで OkiMail アドレスの発行を依頼する
2. OkiMail アドレスを受け取る
3. 外部サービスや検証用途で利用する
4. OkiMail がメールを受信する
5. OkiMail が本文を抽出・デコードする
6. Discord の DM に、本文・メールファイル・添付が転送される

受信は完全自動です。

利用者側で追加設定は不要です。

## 登録完了通知が届かない場合

発行されると、登録完了のお知らせが自動送信で届きます。

DM や年齢設定などの影響で届かない場合があります。

その場合は、OkiMail Bot の「アプリを追加」を押したうえで、発行依頼チャンネルにて再発行を申し出てください。

OkiMail Bot のユーザー ID は `1446422325594554482` です。

## 想定用途

次のような用途を想定しています。

- 外部サービスの検証用
- X などの SNS アカウント作成
- wplace、wiki、サーバー運営に関連する連絡用
- 比較的リスクのあるサイトへの登録
- 各種サービスのログイン通知や障害通知の受信用

これらはあくまで例です。

犯罪などに使用しないでください。

重要な個人情報を扱う用途や、メインのメールアドレスとしての利用は推奨しません。
